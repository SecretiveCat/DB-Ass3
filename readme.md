# Practice Assignment 3

## Завдання 1. Function: Calculate Order Total 

Створено функцію для обчислення загальної вартості замовлення. Функція приймає ідентифікатор замовлення, підсумовує вартість усіх товарів та повертає загальну суму. Якщо замовлення не містить товарів, функція повертає 0 за допомогою функції COALESCE.


## Завдання 2. Procedure: Create New Order 

Реалізовано процедуру для створення нового замовлення. Перед додаванням, спочатку виконується перевірка існування клієнта в таблиці customers. Якщо клієнта не знайдено, тоді висвічується повідомлення про помилку. Якщо перевірка успішна, створюється нове замовлення з поточною датою та початковою сумою 0.


## Завдання 3. Procedure: Add Product to Order  

Створено процедуру для додавання товару до замовлення. Перед виконанням перевіряється, що кількість товару є більшою за нуль та що на складі достатня кількість товару. Після успішної перевірки, товар додається до таблиці order_items, а його залишок на складі автоматично зменшується.


## Завдання 4. Trigger: Update Order Total  

Створено тригер, який автоматично оновлює загальну суму замовлення після додавання, зміни або видалення записів у таблиці order_items. Для перерахунку вартості використовується функція calculate_order_total, що забезпечує актуальність поля total_amount у таблиці orders.


## Завдання 5. Trigger: Order Audit Log 

Реалізовано тригер для ведення журналу створених замовлень. Після додавання нового запису до таблиці orders тригер автоматично створює відповідний запис у таблиці order_log, зберігаючи ідентифікатор замовлення, клієнта, тип дії та дату її виконання.
